# Android Client

This directory will contain the native Android application (Kotlin + Jetpack Compose).

## Status

Not yet scaffolded. The macOS server is being built first.

## Planned Stack

- Language: Kotlin
- UI: Jetpack Compose (Material3)
- Min SDK: API 29 (Android 10)
- Architecture: MVVM
- Network: Raw sockets (UDP + TCP)

## Design

- Dark futuristic HUD theme
- Landscape-primary orientation
- Two main surfaces: full keyboard + trackpad
